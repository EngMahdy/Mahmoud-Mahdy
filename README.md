# نظام إدارة المستندات والمراسلات

تطبيق ويب عربي بالكامل (RTL) لإدارة المستندات والمراسلات بنمط داكن أنيق.

## المميزات

- واجهة عربية كاملة بنظام RTL
- نمط داكن أنيق بخلفية كحلية (#0f172a)
- شريط جانبي بـ 8 أقسام مختلفة
- إضافة وتحرير وحذف المستندات
- تصدير/استيراد البيانات بصيغة JSON
- طباعة فهرس المستندات
- حفظ البيانات محلياً (localStorage)
- واجهة مستجيبة لجميع الشاشات

## الأقسام

1. المحل 21 – مطعم مكبوس حاشي
2. المحل 22 – مؤسسة المملكة للتمور
3. المحل 25 – محمصة حلى الشام
4. الدفاع المدني بالمبنى
5. صيانة
6. شركة التأمين
7. مراسلات عامة
8. مرفقات

## الاستخدام

افتح ملف `index.html` في المتصفح للبدء في استخدام النظام.
